🔧 Firebase Setup Required

Follow these steps to activate your platform:

  1. Go to Firebase Console
  2. Create a new project (name it "VaultPro")
  3. Click on "Web" icon to add a web app
  4. Copy the firebaseConfig object
  5. Replace the config in the code
  6. In Firebase Console, go to Build → Firestore Database
  7. Click "Create database" → Start in test mode
  8. Refresh this page
